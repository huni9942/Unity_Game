영상 외 추가 기능 및 수정점
1. 플레이어 Sprite 및 Animation 변경
2. 몬스터 Sprite 및 Animation 변경 및 추가
3. 무기 추가(기관총) 및 관련 스크립트 수정(동일 Hand 무기 미등장)
4. Start 씬 추가
5. UI 추가, 삭제 및 변경(씬 이동 버튼 추가, Level Up 시 기관총 선택 버튼 추가, 그 외 UI 스프라이트 변경)
6. 레벨 업 시, 특정 아이템만 나오는 버그 수정(foreach문을 추가하여 최대 레벨이 아닌 상태의 아이템이 활성화 되도록 변경)
7. 몬스터 사망 시, 시체가 사라지도록 변경(Invoke문 사용)
8. 몬스터 사망 시 경험치 획득 -> 몬스터 사망 시 경험치 동전 드랍(동전에 충돌 시 경험치 증가)
9. 자석 추가(획득 시 경험치 동전이 캐릭터의 위치로 이동)
10. 자석 버그 수정(미이동, 순간이동, 이동 중 멈춤, 일부만 이동 등)
